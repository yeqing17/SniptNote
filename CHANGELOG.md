# 更新日志

所有项目的显著变更都将记录在此文件中。

## [0.3.0] - 2025-11-20

### ⚡ 性能优化
- **Markdown 渲染优化**：重构 Markdown 解析逻辑，引入独立组件与缓存机制，大幅减少列表滚动与更新时的重复渲染开销。
- **搜索防抖**：为搜索框添加 300ms 防抖，避免输入时的无效计算与界面闪烁。

### 💾 数据持久化
- **本地文件存储**：桌面端数据存储从 `localStorage` 迁移至系统 `AppData` 目录下的 `commands.json` 文件，数据更安全、易备份。
- **自动迁移**：首次启动时自动检测并迁移旧版 `localStorage` 数据，无缝升级。
- **权限修复**：修复 Tauri v2 `fs` 插件权限配置，解决无法写入文件的 Bug。

### 🎨 界面升级
- **视觉重构**：全面升级 UI 设计系统，采用 Slate 色系，提升质感。
- **深色模式修复**：彻底解决深色模式下卡片背景对比度不足（白底白字）的问题，现在卡片拥有独立的深色背景与悬浮阴影。
- **主题完善**：优化 `Ocean`, `Emerald`, `Rose`, `Amber` 等所有主题的配色方案，确保在深色模式下的可读性与美观度。
- **卡片设计**：移除生硬边框，改用现代化的阴影层级设计；优化代码块背景色，增强层次感。

### 🐛 问题修复
- 修复了 Tauri `fs` 插件权限配置错误导致的无法保存数据问题。
- 修复了部分主题变量缺失导致的样式回退问题。

## [0.2.0] - 2025-10-02

- 统一的颜色方案，包括按钮和文本颜色

- 增强的导入逻辑，支持数组结构
- 字段别名支持
- 命令去重功能
- 优化文本显示和对比度

### 修复
- 修复了主题切换时的样式不一致问题
